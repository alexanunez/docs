---


## **Supported Authentication Methods**[**​**](http://localhost:3000/Integrations/Tools/authenticating_tools#supported-authentication-methods)

1. **Bearer Authentication**
2. **Basic Authentication**
3. **API Key-Based Authentication** (via query parameters or headers)
4. **OAuth Authentication** (e.g., for Microsoft, X)
